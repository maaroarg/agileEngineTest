### Installation
Nodejs 9.x (or higher) environment is required

```bash
git clone https://github.com/maaroarg/agileEngineTest
npm install
```

### Getting started

You can start each module separately

```bash
npm run server
npm run http-server
```
... or you can start both!

```bash
npm start
```

### API
http://localhost:3000/transactions/ [GET/POST]
